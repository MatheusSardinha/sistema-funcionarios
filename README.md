# Sistema de Funcionários (CRUD)

Aplicação web desenvolvida para gerenciamento de funcionários, permitindo **criar, visualizar, editar e excluir registros** de forma simples e eficiente.

Fiz esse projeto com foco em aplicar conceitos fundamentais de desenvolvimento web e lógica de programação, simulando um sistema administrativo real.

---

## Funcionalidades

- Cadastro de novos funcionários  
- Listagem dinâmica em tabela  
- Edição de dados existentes  
- Remoção de funcionários  
- Formatação de salário em moeda (BRL)  
- Formatação de data no padrão brasileiro  
- Interface responsiva básica  
- Indicação de modo de edição  

---

## Conceitos aplicados

- Manipulação de DOM com JavaScript
- Separação de responsabilidades (lógica vs interface)
- Estrutura de dados em memória (array de objetos)
- CRUD completo (Create, Read, Update, Delete)
- Eventos e formulários
- Formatação de dados (`toLocaleString`)

---

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript

Projeto

## Como executar o projeto

1. Clone o repositório:

git clone https://github.com/MatheusSardinha/sistema-funcionarios.git

2. Acesse a pasta:

cd sistema-funcionarios

3. Abra o arquivo `index.html` no navegador
